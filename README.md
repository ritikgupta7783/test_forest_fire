# 🔥 Forest Fire Prediction System

## 📌 Overview
Machine Learning model to predict forest fire risk based on environmental features.

## 🚀 Tech Stack
- Python
- Flask
- Scikit-learn
- Pandas, NumPy


## 📊 Features
- Data preprocessing
- Model training (Ridge Regression)
- Web interface for predictions

pip install -r requirements.txt
